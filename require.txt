```markdown
**Q: Flow 是免费的吗？**

**Q: 我的数据安全吗？**
Flow 使用银行级加密技术保护你的财务数据。所有传输使用 HTTPS 加密，数据库采用 AES-256 加密存储。我们绝不向第三方出售你的数据。

**Q: 可以导入其他记账软件的数据吗？**
支持导入标准 CSV 格式的数据。进入"设置" → "数据管理" → "导入数据"，按照模板格式上传即可。

**Q: 支持多人共用账户吗？**
Pro 会员支持创建家庭组，最多 5 名成员共享记账空间，适合家庭或室友共同管理开支。

**Q: 如何联系客服？**
```

### 4.1 恩格尔系数计算
```javascript
function calculateEngelCoefficient(transactions) {
  const foodExpenses = transactions
    .filter(t => t.category === 'food')
    .reduce((sum, t) => sum + t.amount, 0);
  
  const totalExpenses = transactions
    .filter(t => t.type === 'expense')
    .reduce((sum, t) => sum + t.amount, 0);
  
  return totalExpenses > 0 ? (foodExpenses / totalExpenses).toFixed(2) : 0;
}

// 解读：>0.6 贫困, 0.5-0.6 温饱, 0.4-0.5 小康, 0.3-0.4 富裕, <0.3 最富裕
function getEngelStatus(coefficient) {
  if (coefficient > 0.6) return { level: '贫困', color: '#FF3B30' };
  if (coefficient > 0.5) return { level: '温饱', color: '#FF9500' };
  if (coefficient > 0.4) return { level: '小康', color: '#FFCC00' };
  if (coefficient > 0.3) return { level: '富裕', color: '#34C759' };
  return { level: '最富裕', color: '#007AFF' };
}
```

### 4.2 预算提醒逻辑
```javascript
// 检查预算并触发提醒
function checkBudgetAlert(userId, category, amount) {
  const budget = getMonthlyBudget(userId, category);
  const spent = getMonthlySpent(userId, category);
  const newSpent = spent + amount;
  const ratio = newSpent / budget;
  
  if (ratio > 0.9) {
    return { type: 'danger', message: `您的${category}预算即将超支（${(ratio*100).toFixed(0)}%）` };
  } else if (ratio > 0.8) {
    return { type: 'warning', message: `您的${category}预算已使用 ${(ratio*100).toFixed(0)}%` };
  }
  return null;
}
```
